# 玩世不恭

不要问我做了什么工作  
不要问我出门坐什么车  
不要问我如何去生活  
不要问我女朋友是哪一个  
有人说三十后才明白  
二十岁到底要我做什么  
不要怪我无从去选择  
看一看现在的生活  
七十岁的爷爷领导中华  
六十岁的伯伯干四化  
五十岁的叔叔溜溜达达  
四十岁的老哥还想发家  
有些人每天他忙改革  
有些人每天在忙出国  
有些人每天他忙赌博  
有些人没有工作  
忘掉吧 沉重的工作  
忘掉吧 爱情的折磨  
忘掉吧 金钱的诱惑  
忘掉吧 复杂的交通规则  
不要管别人在做什么  
我要过属于自己的生活  
不要管别人在说什么  
我只要尽情欢唱
