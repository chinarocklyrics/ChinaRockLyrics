# 不用再多说

突然间 我厌倦  
厌倦自己的欺骗  
所有笑容都已凝结  
剩下的只有时间  
我哭过 我笑过  
最后却只有一个结果  
告诉我天堂的情歌  
到底在哪里失落  
我记得 我说过  
去经受所有的折磨  
尝尽人间的滋味  
现在却只剩下沉睡  
唤醒我 抓紧我  
别让我失去这一刻  
我们已浪费得太多  
什么话都不用说  
给我沉默  
给我烦恼  
给我骄傲  
给我一点微笑  
给我风暴  
给我知道  
给我尽头  
给我一点自由
