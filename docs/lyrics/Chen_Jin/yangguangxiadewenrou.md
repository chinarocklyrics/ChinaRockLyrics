# 阳光下的温柔

我在黑暗之中  
看着北斗七星  
距离那么遥远  
心中感到一种难受  
黑夜使我祈求  
霓虹之夜破旧  
擦亮我的皮鞋  
寻找阳光下的温柔  
我看到太多的诱惑欺骗  
看到被害的人在我面前  
我在喊  
我在喊  
有谁听见  
来牵着我无助的双手  
带着晚霞的天空  
回到没有邪恶的梦里  
在音乐中
