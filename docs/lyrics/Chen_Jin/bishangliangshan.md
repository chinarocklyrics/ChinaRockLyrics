# 逼上梁山

话说从前  
水浒梁山  
山上住着是一群好汉  
替天行道  
聚义揭竿  
扫除人间的恶和难  
反压迫  
反贪官  
不受欺骗  
不受招安  
百姓称好  
官府难安  
其实他们都是逼上梁山  
哦 梁山  
哦 好汉  
哦 造反
