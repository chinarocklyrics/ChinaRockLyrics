# 世纪末之梦

灵魂从死人堆里站起  
爬上了天和地  
恶魔侵占晴朗天空  
黑暗在地球上拥挤  
太阳躲进了骚乱的云雾深底  
死神降临废墟瓦砾  
暴戾张开了锋利的血牙  
天崩地裂充满杀气  
狂风暴雨冲打着大地  
荡平了山川屋脊  
邪恶喷吐熊熊火焰  
疯狂吞噬精神肉体  
少女和丑恶残忍的野兽在一起  
羞耻贞洁面目奇异  
智慧和妖魔紧紧地抱在一起  
摧毁文明灭掉世纪  
我们都是复活的魔鬼  
疯狂的野性是我们的武器  
我们都是复活的魔鬼  
控制了海洋和山川陆地  
嘿呦嘿  
复活的魔鬼  
主宰这世界  
我们是复活的魔鬼
