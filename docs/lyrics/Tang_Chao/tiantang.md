# 天堂

总是感觉匆匆忙忙  
想法太多  
希望太少  
岁月反复无常  
过去太遥远  
未来太迷茫  
时间在那梦里躲藏  
失眠的恐慌  
奔跑的欢畅  
在麻醉和迷幻里徜徉  
那些来去匆匆姑娘  
带我走进快乐梦乡  
却使我越来越习惯
