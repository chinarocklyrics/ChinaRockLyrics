# 同在一片天空下

仰望着天  
心一片茫然  
看不到往日蔚蓝  
所有欢笑  
所有的幻想  
一切都将不存在  
炮声在笑  
那天空在燃烧  
绿色的城市已失去了往日的喧闹  
美好的世界  
变得如此的凄凉  
残破的肢体上  
充满着痛苦悲伤  
炮声在唱  
那大地在燃烧  
恐惧的人群在火光之中到处奔跑  
悲伤的脸上失去往日的笑  
不再歌唱  
只有默默祈祷  
同在一片天空下成长

曾经共有同一个幻想
