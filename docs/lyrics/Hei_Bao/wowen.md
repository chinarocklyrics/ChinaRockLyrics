# 我问

我问什么付出太多以后  
总让我太失望  
我问何时能够推起天窗  
推倒四周的墙  
我知道你会用善解人意的目光  
我看到黑夜里你也会  
脱掉那身扮装  
我们一起看过一盏烛光  
你却擦它的脏  
为何能够走过漫漫长夜  
却怕黎明太阳  
我想要用你的善良缝合我的伤  
我知道透明的躯体里  
已经没有力量  
难道总想让我不问不想不忙  
你才感到我好  
我在天空之上  
看到风吹你的臂膀左右摇晃  
难道你看不到墙壁四周的目光  
如果你推开窗撕去装  
就会看到光芒把你照亮
