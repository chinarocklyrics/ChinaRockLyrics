# 这儿的空间

打不开天 也穿不过地  
那自由不过不是监狱  
你离不开我 我也离不开你  
谁都不知到底是爱还是赖  
钱就是钱 利就是利  
你我不过不是奴隶  
你只能为了我 我也只能为了你  
不过不是一对一对儿虾米  
这儿的空间 没什么新鲜  
就像我对你的爱情里没什么秘密  
我看着你 曾经看不到底  
谁知进进出出才明白是无边的空虚  
就像这儿的空间里
