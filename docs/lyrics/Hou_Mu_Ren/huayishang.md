# 花衣裳

我要爬到最高的山上  
去寻找迷人的风光  
我要游到最深的海洋  
去寻找很久的梦想  
我要走到最远的地方  
去寻找那属于我的家乡  
我唱着最动听的歌谣  
为了你 爱我的姑娘  
哦 我的姑娘  
请你跟我一起去看好风光  
哦 我的姑娘  
你看那海鸥在飞翔  
哦 我的姑娘  
请你穿上你的花衣裳  
哦 姑娘 姑娘  
哦 请你呆在我的身旁  
哦 你的头发又黑又亮
